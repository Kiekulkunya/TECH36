🚀 Project Overview
This project explores "Vibe Coding"—using natural language and AI assistants (Gemini, ChatGPT, and Claude) to clean data, perform EDA, and build a functional retail analytics dashboard.

1. Data Preparation & Exploration
Data Source: Retail Intelligence 100K User Behavior Dataset (Kaggle)
(www.kaggle.com/datasets/noopurbhatt/retail-intelligence-100k-user-behavior-dataset/data?select=retail_user_behavior_100k.csv)

Cleaning & EDA: Utilized Claude for data cleaning. View Cleaned Dataset
 (https://github.com/Kiekulkunya/TECH36/blob/main/retail_user_behavior_100k_clean.csv)

Comparative Analysis: Asked Gemini, ChatGPT, and Claude to analyze original vs. clean files to understand data structure and quality.

Prompt Engineering: Queries generated via Claude for dashboard creation. View Prompts
(https://github.com/Kiekulkunya/TECH36/blob/main/Prompts%20for%20assignment%201.txt)

4.6 10-Bullet Highlights: 
    - Analyzed a 108k-row retail dataset to identify key conversion drivers and drop-off points.
    - Built a D3.js dashboard with dark/light themes featuring a conversion funnel, regional heatmap, and source distribution.
    - Implemented global state management allowing category filters to update all visualizations simultaneously.
    - Added cross-chart interactivity where clicking a region or device segment filters the entire dashboard view.
    - Integrated a Papa Parse CSV uploader for dynamic data processing with schema validation and loading states.
    - Deployed an AI Analyst panel powered by Gemini 1.5 Flash for contextual Q&A based on filtered dataset summaries.
    - Added a Gemini TTS feature to narrate AI-generated insights using synthesized professional voices.
    - Created a real-time product table that displays top items specifically for the currently hovered funnel stage.
    - Drafted advanced JavaScript logic for Sankey diagrams, Chord diagrams, and regional price elasticity lines.
    - Applied a UX executive-level critique to improve revenue visibility, data honesty, and mobile channel alignment.

Key takeaways: 
  1. Fast and Effective building prototype or mockup with less details.
  2. Very difficult to debug of errors
  3. Poor Memory
  4. Good to learn but Gemini does not provide sufficient quality of application or dashboard creation
